# Nectar
price comparison website with flask. 

## Cloning the Website
1. Clone the repo.
```
git clone https://github.com/andrearcaina/Nectar.git
```
2. Change into the repo directory.
```
cd Nectar
```
3. Switch to the branch that you are working on by using:
```
git checkout <branch-name>
```
